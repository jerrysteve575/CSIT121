Reflection

This project helped me understand how to build cohesive and accessible websites. I learned how to create semantic HTML, apply responsive CSS, and manage a project layout across multiple pages.

Testing on both Chrome and Firefox showed that some margin/padding differences can affect layout, especially when using flexbox. My mobile device test revealed that using @media queries is essential to keep content readable on smaller screens.

Overall, this course strengthened my HTML/CSS skills and helped me learn how to organize a full site from scratch.
